Interface de Gestion de Données Excel avec Tkinter

Objectif : Cette application de bureau utilise Tkinter, la bibliothèque d'interface graphique pour Python, pour offrir une solution élégante et fonctionnelle de gestion de données. Elle est conçue pour permettre aux utilisateurs de :
Importer un Fichier Excel : Sélectionnez facilement un fichier Excel depuis votre système.
Afficher les Données : Visualisez le contenu du fichier dans un tableau structuré et facilement navigable.

Technologies Utilisées :

  • Python
  • Tkinter
  • Pandas

Fonctionnalités :

  • Sélection Facile de Fichiers : Interface utilisateur intuitive pour sélectionner et charger des fichiers Excel.
  • Affichage Dynamique : Les données sont affichées dans un tableau interactif qui ajuste automatiquement ses colonnes en fonction du contenu du fichier.
  • Interface Moderne : Design épuré avec une palette de couleurs contemporaines pour une meilleure expérience utilisateur.
  • Gestion des Données : Suppression automatique des lignes vides pour une vue propre des données pertinentes.
